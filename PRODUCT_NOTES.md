# AthLink Hub Product Notes

## V1 Positioning
AthLink Hub helps parents in Atlantic Canada find local support for young athletes, including coaching, training, recovery, gear, camps, and facilities.

## Launch Region
- Newfoundland & Labrador
- Nova Scotia

## V1 Features
- Parent search
- Provider profiles
- Save providers
- Add business
- Claim listing
- Admin review dashboard
- AI-discovered provider queue
- External booking/contact links

## Not in V1
- Payments
- Full in-app scheduling
- Reviews
- Messaging
- Mobile app store release
